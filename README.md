# 7. アカウントIDの設定
アカウントのアイコンを押し、「settings」にアクセスしてください。
「Account」タブの「Change username」項目の「Change username」ボタンを押下して、アカウントIDを設定してください。</br>
※アカウントIDは「TSWG-社員番号」にしてください。
